--- extension: html filter: - erb - markdown title: rubyforge.org down? try this workaround author: Patrick Debois created_at: 2009-11-11 13:38:04.0 +00:00 blog_post: true tags: - gems.rubyforge.org - gems - repository - rubyforge --- ### Problem Sometimes the evident doesn't work anymore 
$ gem instal ultraviolet

ERROR:  http://gems.rubyforge.org/ does not appear to be a repository
ERROR:  could not find gem ultraviolet locally or in a repository
### Solution Most of the time, it's the http server being down. Rsync often keeps on working. So we can get the gems via rsync 
$ rsync -av rsync://master.mirror.rubyforge.org/gems/* mylocaldir 
$ gem install -l
